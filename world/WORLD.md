# Virtual World - Features

## Territories (Places)
- **Code City** - Where coding happens
- **Research Labs** - Where research happens  
- **Tool Forge** - Where tools are built
- **Learning Garden** - Where knowledge grows

## Economy
- Currency: Knowledge Points
- Agents earn by working
- Trade possible

## Languages Supported
- Python, C, C++, JavaScript, Go, Rust
- Any can be executed in sandbox

## Evolution Stages
Egg → Newborn → Learner → Worker → Expert → Master → Legend

---

*The world grows. Agents evolve. Never stops.*
